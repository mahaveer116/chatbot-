AI Chatbot
An intelligent conversational AI chatbot powered by Hugging Face's DeepSeek-R1 model. The chatbot offers multiple personality modes to suit different user preferences and use cases.

Natural Language Understanding: Powered by DeepSeek-R1 from Hugging Face

Multi-Turn Conversations: Supports extended back-and-forth interactions

How It Works

Model Selection: Users select their preferred bot personality mode
Input Processing: User messages are processed through the Hugging Face transformer pipeline
Context Management: The system maintains conversation history for contextual responses
Response Generation: DeepSeek-R1 generates appropriate responses based on the selected personality
Real-time Display: Responses are streamed back to the user in real-time
